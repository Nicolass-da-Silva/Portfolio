## ✨ Visão Geral

Este projeto foi desenvolvido para apresentar minhas habilidades, projetos e experiências de forma clara, interativa e responsiva. A ideia é unir visual atrativo com organização e performance.

---

## 🎓 Origem do Projeto

Este projeto nasceu como um trabalho acadêmico da faculdade, mas acabou se tornando algo maior. O que começou como uma tarefa obrigatória virou um impulso para tirar do papel algo que eu já queria fazer há bastante tempo: um portfólio profissional para expor minhas habilidades e projetos.


## 🚀 Funcionalidades

* 📱 Layout responsivo (desktop, tablet e mobile)
* 🎞️ Carrosséis e componentes dinâmicos
* 📂 Sessão de projetos com cards
* 🧭 Navegação intuitiva
* 🎨 Design moderno e animações suaves
* 🧩 Arquitetura simples e escalável
